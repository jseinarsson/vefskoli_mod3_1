This project is styled with Sass. Sass is a CSS pre-compiler that allows you to code smarter CSS with more oversight. To see an example of how Sass works head over to my github repo and check out the .scss files found in the style/scss folder. There are comments explaining some of the Sass features that I find helpful and used in this project. For more Sass resources, check out the links above. I especially recommend the video on the Module System update, as it explains some new features and how Sass plans to phase out the @import rule by the end of 2022.
